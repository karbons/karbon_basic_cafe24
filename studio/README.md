# 카본빌더 개발 환경

## 개발 환경

웹 gui 환경에서 개발 환경을 구축합니다. 

## 원칙

1. 개발환경과 실제 서비스 코드를 분리해서 최종 서비스 코드는 단순화 히여 수정시 직관적으로 이해할 수 있게 코드를 독립하여 구현합니다.
2. 최종 코드는 사용 목적에 필요한 코드만 남기고 경량이어야하고 최적화되어야 합니다.
3. 개발환경은 개발을 하는 모든 행위를 관리 할 수 있도록 개발 생산관리 erp의 역할을 합니다.
4. 카본빌더는 쉬운 개발이 최종적인 결과물에 복잡함을 초래하면 안된다는 원칙을 갖고 있습니다.
5. 개발환경과 실제 서비스 코드는 분리해서 결과물의 순수성을 보장합니다.
6. 카본빌더는 saas를 지양합니다. 고객이 원하는 모든 것을 직접 개발할 수 있도록 지원합니다.
7. 카본 공유ai는 개발자 pc의 gpu 품앗이 기능입니다.
8. 카본 공유ai는 멀티에이전트를 로컬 llm을 이용하기 위해 연결되어 있는 타 회원의 pc의 유휴 gpu를 활용하는 기능입니다.
9. 카본 개발환경은 개발사가 프로젝트를 수주하고 컨펌을 받고 사후 관리하는 


### 개발 생산관리 erp 기능
- 개발 계획
- 개발 일지
- 데이터베이스 관리 (supabase 방식으로 db를 설계하고 관리합니다.)
- erd 다이어그램 관리
- 변수 모델 및 타입정의
- 디자인 계획(스토리보드, 프로토타입)
- 디자인 컨펌(고객 및 내부 컨펌)
- 디자인 정의(디자인 시스템, 컴포넌트 정의)
- 외주 계약서 관리(계약서 작성 및 관리)
- 개발환경 관리(개발환경, 테스트환경, 운영환경)
- 버전 및 배포 관리(버전 관리, 배포 관리)
- 컴포넌트샵에서 컴포넌트를 가져와서 개발 환경에 추가하고 개발 환경에서 사용할 수 있도록 합니다.
- ai 컨텍스트 관리(코드 컨텍스트, 디자인 컨텍스트)
- 온톨로지 관리(데이터에 대한 온톨로지 관리)
- 개발 수익 관리
- 개발자 품평 관리
- 마케팅 관리
- 접속자 로그 분석
- 하나의 프로젝트 뿐만 아니라 여러 프로젝트를 관리 할 수 있도록 합니다.
- 개발 트렌드 추적
- 서비스 장애 및 에러 로그 분석
- 서비스 개선사항 접수 및 처리
- 업데이트 관리



## 기술명세
1. 개발환경 관리 환경은 sveltekit으로 구축합니다.
2. 카본빌더의 개발환경
- frontend: sveltekit
- backend: 그누보드5(php8.2), rust axum(업그레이드 확장 또는 대체 기능)
- database: mysql, postgresql
- ai: 카본에서 제공하는 공유 ai를 연결합니다.

### 기능 명세
1. 환경변수를 웹 ui로 등록하면 이를 .env 파일로 생성해서 관리합니다.
2. 개발환경을 개발 pc에 docker로 구축하고 관리합니다.
3. 배포는 선택에 따라 github action을 이용하거나 스크립트 실행을 통해 수동으로 배포합니다.
4. 최초 설치는 /dev 폴더를 실행하면 install gui가 실행됩니다. 이 gui를 통해 개발환경을 구축하고 관리합니다.
5. admin user 정보를 넣고 실행을 하면 개발환경이 구축됩니다.
6. db개발환경용 셋트와 그누보드 최신버전을 github에서 다운로드 받아서 설치합니다.
7. 이후 그누보드 폴더에 api 설치를 합니다.
8. 여러 다양한 배포 환경을 json 파일로 저장하고 관리합니다.(배포환경 최적화 세트, 서버 환경 세트 등을 공유해서 회원들이 배포환경을 쉽게 구축할 수 있도록 합니다.)
- cafe24 호스팅용 환경 세트
- aws ec2 환경 세트
- 우리집 pc 환경(윈도우, 우분투) 세트

# karbon studio

카본 스튜디오는 카본빌더를 개발하는 개발 관리 도구입니다.

## 주요 기능
- 카본빌더 개발환경을 자동 구성해줍니다.
- 카본빌더의 개발 현황을 한눈에 볼 수 있습니다.
- 개발의 기획, erd, db, 변수정의, 개발(바이브코딩, AI코딩)까지 통합 관리합니다.
- 개발 에디터를 제공합니다.
- 개발 진행 상황을 관리할 수 있습니다.
- 배포 및 운영을 지원합니다.
- 서버 상태 관리
- 고객 동선 분석 및 개선
- 민원 분석 및 개선
- AI 온톨로지 개발 환경
- 컴포넌트 공유
- AI 공유 API
- 카본빌더는 결과물로 카본 스튜디오의 기능과는 별도로 동작합니다.
- 카본빌더는 순수한 서비스 데이터만으로 구성됩니다. 카본 스튜디오는 개발 관리 도구입니다.

1. 카본빌더

2. 카본 스튜디오
