# [PRD] 그누보드5 현대화: FAPI + Svelte 5 하이브리드 앱 스타터

## 1. 프로젝트 개요

* **제품명:** Karbon FAPI (Simple API) & Svelte 5 앱 템플릿
* **목적:** 그누보드5의 레거시 인프라와 최신 프론트엔드(Svelte 5)를 연결하여, 사용자가 AI를 통해 1시간 이내에 자신의 사이트를 하이브리드 앱으로 전환하게 함.
* **핵심 가치:** "API 개발 몰라도 됨, Svelte 몰라도 됨. AI가 다 해주는 환경을 제공함."

---

## 2. 시장 분석 및 타겟 (Market Context)

* **현황:** 그누보드5 시장은 크지만, 모바일 앱 전환 시 웹뷰(WebView) 수준에 머물러 있음. REST API 부재로 고성능 앱 개발이 난해함.
* **타겟 유저:** 1. 기존 그누보드5 운영자 (쇼핑몰, 커뮤니티).
2. 앱 개발 수주를 받았으나 시간이 부족한 프리랜서.
3. Svelte 5를 배우고 싶지만 기초 세팅이 어려운 초보 개발자.

---

## 3. 핵심 기능 (Features) - 3일 개발 범위

### Phase 1. FAPI Lite (Backend / PHP)

* **단일 파일 설치:** `fapi.php` 하나만 올리면 작동하는 구조.
* **핵심 엔드포인트:**
* `auth`: 그누보드 세션 기반 로그인/로그아웃.
* `board`: 게시판 리스트, 글 읽기, 글 쓰기 JSON 출력.
* `shop`: (영카트용) 상품 리스트 및 상세 정보 호출.


* **보안:** API Key 또는 Referer 체크를 통한 간단한 보안 강화.

### Phase 2. Svelte 5 App Shell (Frontend / SvelteKit)

* **Svelte 5 Runes 적용:** `$state`, `$effect`를 활용한 데이터 바인딩 샘플.
* **Capacitor 세팅:** Android/iOS 빌드 환경 즉시 사용 가능하도록 설정.
* **UI 샘플:** 그누보드 기본 게시판을 앱 스타일(당근마켓형)로 출력하는 샘플 페이지.

### Phase 3. AI 개발용 컨텍스트 (The Moat)

* **`AI_CONTEXT.md` 제공:** * FAPI 데이터 구조 명세.
* Svelte 5 룬(Runes) 사용 규칙.
* Cursor/ChatGPT용 "내 게시판 추가하기" 프롬프트 템플릿.



---

## 4. 시장 분석 툴 기획 (트렌드 분석용)

개발 우선순위를 정하기 위해 1일차에 실행할 분석 모듈입니다.

* **기능:** 1. SIR 컨텐츠몰의 카테고리별 판매량/조회수 스크래핑.
2. '예약', '결제', '채팅' 등 키워드 빈도수 분석.
3. 최근 6개월간 인기 있는 스킨의 UI 스타일(테일윈드 여부 등) 분류.

---

## 5. 개발 일정 (3-Day Sprint)

| 일정 | 단계 | 세부 작업 | 비고 |
| --- | --- | --- | --- |
| **Day 1** | **시장분석 & FAPI** | SIR 트렌드 분석 + PHP 기반 FAPI 핵심 로직 개발 | 시장 니즈 확정 |
| **Day 2** | **Svelte 5 템플릿** | SvelteKit + Capacitor 연동 + 룬(Runes) 기반 샘플 UI 제작 | 앱 성능 최적화 |
| **Day 3** | **패키징 & 배포** | AI 가이드 문서 작성 + 교육 영상(짧은 튜토리얼) + 컨텐츠몰 등록 | 수익화 시작 |

---

## 6. 미래 확장성 (Roadmap)

1. **Firebase 연동:** 실시간 채팅 및 푸시 알림 모듈 추가.
2. **노코드 빌더:** 관리자 페이지에서 컴포넌트 순서를 바꾸는 JSON 빌더.
3. **특화 빌더:** 주식/코인 대시보드 및 테이블 오더 시스템 전용 테마 출시.

---

## 7. 성공 지표 (KPI)

* SIR 컨텐츠몰 등록 후 1주일 내 판매 건수 5건 이상.
* 구매자 문의 중 "설치가 쉽다"는 피드백 80% 이상.
* AI 가이드 활용을 통한 사용자 커스텀 성공 사례 발생.
