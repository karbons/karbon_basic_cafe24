# 그누보드5 메인 사이트 기본 기능 목록

## 1. 개요

이 문서는 그누보드5 메인 사이트(관리자가 아닌 일반 사용자용)의 기본 기능들을 정리한 문서입니다. FAPI 개발 시 필요한 기능들을 파악하기 위한 참고 자료입니다.

## 2. 회원 관련 기능

### 2.1 회원 인증

#### 2.1.1 로그인
- **파일:** `bbs/login.php`
- **기능:**
  - 일반 로그인 (아이디/비밀번호)
  - 소셜 로그인 (확장 가능)
  - 자동 로그인 (쿠키 기반)
  - 로그인 실패 횟수 제한
  - 차단/탈퇴 회원 로그인 방지
  - 메일 인증 확인

#### 2.1.2 로그아웃
- **파일:** `bbs/logout.php`
- **기능:**
  - 세션 삭제
  - 쿠키 삭제
  - 로그인 정보 초기화

#### 2.1.3 회원가입
- **파일:** `bbs/register.php`, `bbs/register_form.php`
- **기능:**
  - 회원 정보 입력
  - 아이디 중복 확인
  - 이메일 중복 확인
  - 닉네임 중복 확인
  - 본인인증 (선택)
  - 약관 동의
  - 메일 인증 발송

#### 2.1.4 회원정보 수정
- **파일:** `bbs/register_form.php` (w=u 모드)
- **기능:**
  - 기본 정보 수정
  - 비밀번호 변경
  - 프로필 이미지 변경
  - 본인인증
  - 성인인증

#### 2.1.5 회원 탈퇴
- **파일:** `bbs/member_leave.php`
- **기능:**
  - 탈퇴 처리
  - 개인정보 삭제/익명화
  - 작성글 처리 옵션

#### 2.1.6 비밀번호 찾기
- **파일:** `bbs/password_lost.php`, `bbs/password_lost2.php`
- **기능:**
  - 아이디/이메일로 찾기
  - 임시 비밀번호 발송
  - 비밀번호 재설정

#### 2.1.7 본인인증
- **파일:** `bbs/member_cert_refresh.php`
- **기능:**
  - 본인인증 갱신
  - 성인인증

### 2.2 회원 서비스

#### 2.2.1 쪽지
- **파일:** `bbs/memo.php`, `bbs/memo_form.php`, `bbs/memo_view.php`
- **기능:**
  - 쪽지 보내기
  - 받은 쪽지
  - 보낸 쪽지
  - 쪽지 읽기
  - 쪽지 삭제

#### 2.2.2 스크랩
- **파일:** `bbs/scrap.php`, `bbs/scrap_popin.php`
- **기능:**
  - 게시글 스크랩
  - 스크랩 목록
  - 스크랩 삭제

#### 2.2.3 포인트
- **파일:** `bbs/point.php`
- **기능:**
  - 포인트 내역 조회
  - 포인트 적립/차감 내역

#### 2.2.4 프로필
- **파일:** `bbs/profile.php`
- **기능:**
  - 회원 프로필 조회
  - 작성글 목록
  - 작성 댓글 목록

## 3. 게시판 관련 기능

### 3.1 게시판 목록

#### 3.1.1 게시글 목록 조회
- **파일:** `bbs/list.php`
- **기능:**
  - 게시글 목록 출력
  - 공지글 표시
  - 카테고리 필터링
  - 검색 기능
  - 정렬 기능 (날짜, 조회수, 추천 등)
  - 페이징

#### 3.1.2 게시글 상세 조회
- **파일:** `bbs/view.php`
- **기능:**
  - 게시글 내용 출력
  - 조회수 증가
  - 이전/다음글 링크
  - 첨부파일 다운로드
  - 권한 체크 (읽기, 수정, 삭제)
  - 비밀글 처리
  - 본인인증/성인인증 체크

### 3.2 게시글 작성/수정

#### 3.2.1 게시글 작성
- **파일:** `bbs/write.php` (w='')
- **기능:**
  - 게시글 작성 폼
  - 에디터 (DHTML 에디터 지원)
  - 파일 첨부
  - 링크 추가
  - 카테고리 선택
  - 비밀글 설정
  - HTML 사용 설정
  - 이메일 알림 설정
  - 캡차 (비회원)

#### 3.2.2 게시글 수정
- **파일:** `bbs/write.php` (w='u')
- **기능:**
  - 기존 게시글 수정
  - 권한 체크
  - 답변글 있는 원글 수정 제한
  - 댓글 수 제한에 따른 수정 제한

#### 3.2.3 게시글 삭제
- **파일:** `bbs/delete.php`
- **기능:**
  - 게시글 삭제
  - 권한 체크
  - 비밀번호 확인 (비회원)
  - 첨부파일 삭제

#### 3.2.4 답변글 작성
- **파일:** `bbs/write.php` (w='r')
- **기능:**
  - 답변글 작성
  - 답변 단계 제한 (최대 10단계)
  - 답변 개수 제한 (최대 26개)

#### 3.2.5 게시글 이동/복사
- **파일:** `bbs/move.php`
- **기능:**
  - 게시글 이동
  - 게시글 복사
  - 관리자 권한 필요

### 3.3 댓글 기능

#### 3.3.1 댓글 조회

## 4. 최신글 기능

### 4.1 최신글 출력

#### 4.1.1 최신글 함수
- **파일:** `lib/latest.lib.php`
- **함수:** `latest($skin_dir, $bo_table, $rows, $subject_len, $cache_time, $options)`
- **기능:**
  - 지정된 게시판의 최신글 출력
  - 스킨별 커스터마이징
  - 캐시 지원
  - 제목 길이 제한

#### 4.1.2 사용 예시
```php
// 기본 최신글
latest('basic', 'free', 10, 40);

// 사진 최신글
latest('pic_list', 'gallery', 4, 23);

// 테마 스킨 사용
latest('theme/pic_list', 'notice', 4, 23);
```

#### 4.1.3 최신글 캐시
- **파일:** `lib/cache.lib.php`
- **함수:** `g5_latest_cache_data()`
- **기능:**
  - 최신글 데이터 캐싱
  - 성능 최적화

## 5. 메뉴 기능

### 5.1 메뉴 출력

#### 5.1.1 메뉴 함수
- **파일:** `lib/get_data.lib.php`
- **함수:** `get_menu_db($me_code, $is_admin)`
- **기능:**
  - 메뉴 데이터 조회
  - 계층 구조 지원 (1차, 2차 메뉴)
  - PC/모바일 구분
  - 관리자 메뉴 포함 여부

#### 5.1.2 메뉴 구조
- **테이블:** `g5_menu`
- **컬럼:**
  - `me_id`: 메뉴ID
  - `me_code`: 메뉴코드
  - `me_name`: 메뉴명
  - `me_link`: 링크
  - `me_target`: 링크 타겟
  - `me_order`: 정렬순서
  - `me_use`: 사용여부
  - `me_mobile_use`: 모바일 사용여부

#### 5.1.3 메뉴 출력 예시
```php
// PC 메뉴
$menu_datas = get_menu_db(0, true);
foreach($menu_datas as $row) {
    echo $row['me_name'];
    foreach($row['sub'] as $sub) {
        echo $sub['me_name'];
    }
}

// 모바일 메뉴
$menu_datas = get_menu_db(1, true);
```

## 6. 배너 기능

### 6.1 배너 출력

#### 6.1.1 배너 함수 (쇼핑몰)
- **파일:** `lib/shop.lib.php`
- **함수:** `display_banner($position, $skin)`
- **기능:**
  - 지정된 위치의 배너 출력
  - 스킨별 커스터마이징
  - PC/모바일 구분
  - 기간별 노출 제어

#### 6.1.2 배너 구조
- **테이블:** `g5_shop_banner`
- **컬럼:**
  - `bn_id`: 배너ID
  - `bn_position`: 배너 위치
  - `bn_device`: 기기 구분 (both/pc/mobile)
  - `bn_begin_time`: 시작일시
  - `bn_end_time`: 종료일시
  - `bn_order`: 정렬순서
  - `bn_url`: 링크 URL
  - `bn_alt`: 대체 텍스트
  - `bn_border`: 테두리 여부
  - `bn_new_win`: 새창 여부

#### 6.1.3 배너 사용 예시
```php
// 메인 배너
display_banner('메인', 'mainbanner.10.skin.php');

// 사이드 배너
display_banner('왼쪽', 'boxbanner.skin.php');
```

### 6.2 배너 클릭 추적
- **파일:** `shop/bannerhit.php`
- **기능:**
  - 배너 클릭 카운트
  - 리다이렉트

## 7. 팝업 기능

### 7.1 팝업 출력

#### 7.1.1 팝업 파일
- **파일:** `bbs/newwin.inc.php` (PC), `mobile/newwin.inc.php` (모바일)
- **기능:**
  - 팝업 레이어 출력
  - 기간별 노출 제어
  - PC/모바일 구분
  - 커뮤니티/쇼핑몰 구분
  - 쿠키 기반 재노출 방지

#### 7.1.2 팝업 구조
- **테이블:** `g5_new_win`
- **컬럼:**
  - `nw_id`: 팝업ID
  - `nw_division`: 구분 (both/comm/shop)
  - `nw_device`: 기기 구분 (both/pc/mobile)
  - `nw_begin_time`: 시작일시
  - `nw_end_time`: 종료일시
  - `nw_disable_hours`: 비활성화 시간
  - `nw_left`: 왼쪽 위치
  - `nw_top`: 위쪽 위치
  - `nw_width`: 너비
  - `nw_height`: 높이
  - `nw_content`: 내용

#### 7.1.3 팝업 사용 예시
```php
// head.php에 포함
include_once(G5_BBS_PATH.'/newwin.inc.php');
```

## 8. 내용관리 기능

### 8.1 내용 출력

#### 8.1.1 내용관리 함수
- **파일:** `lib/get_data.lib.php`
- **함수:** `get_content_db($co_id)`, `get_content_by_field()`
- **기능:**
  - 내용 데이터 조회
  - SEO 제목으로 조회
  - HTML 변환
  - 변수 치환 ({{회사명}}, {{대표자명}} 등)

#### 8.1.2 내용 구조
- **테이블:** `g5_content`
- **컬럼:**
  - `co_id`: 내용ID
  - `co_html`: HTML 사용여부
  - `co_subject`: 제목
  - `co_content`: 내용
  - `co_seo_title`: SEO 제목
  - `co_mobile_content`: 모바일 내용
  - `co_tag_filter_use`: 태그 필터 사용

#### 8.1.3 내용 사용 예시
```php
$co = get_content_db('about');
echo $co['co_subject'];
echo conv_content($co['co_content'], $co['co_html']);
```
